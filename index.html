Linux下nohup命令的使用

Posted on 2020-02-16

nohup是之前在写一个基于python的服务器部署时用到的，服务器一部分是基础的网站后台开发，使用Django作为框架接收前端请求，用的Ubuntu+Nginx+Django的部署方案，另一部分和嵌入式通过socket进行数据传输，数据库读写 （现在想来是一个生产-消费者模式，这部分功能的实现我写成了几个python脚本，燃鹅服务器并不像平常在PC上使用的Linux，没有图形界面更不能开好几个终端。为了让它在后台持续不断地运行并且不占用当前终端，在网上查找了很多信息，最终用nohup命令解决了这个问题。

nohup即no hang up,不挂断地运行
使用nohup Command [ Arg … ]如 python test.py并无在后台运行地功能，在命令后面加上&即可

输出默认重定向到nohup.out
由于我程序里面写的是绑定一个端口并监听，可以用lsof -i:端口号查看端口占用情况
其余常用的命令：
jobs -l:查看当前终端生效的进程
kill -9 进程号:杀死进程
ps -ef:显示所有进程信息

《活着》

Posted on 2019-11-21 Edited on 2019-11-22 In 读书笔记

是在一个没课的上午伴着一杯咖啡安安静静看完的这本书

读这本书的过程给我最直观的感受就像看了一部电影，书中老人以第一人称视角平淡地讲述了自己在外人看来很悲惨地一生。

从早期的吃喝嫖赌，到后面赔光家产受尽屈辱，为了生存竭尽全力，期间亲人相继离开，朴实的讲述里隐藏着那段岁月不懂声色的力量。

我也时常在思考，人活着的意义在哪里，什么才是值得去追求的。看完这本书更像是在短短几个小时里匆匆扮演了别人人生的一个旁观者，在略微清闲安稳的现下感受到生命的无常与悲苦。

“生活是属于每个人自己的感受，不属于任何别人的看法。”

我承认，20岁的年纪，我做不到完全不在意别人的看法，我一直标榜自己追求滚烫的人生，很长的一段时间里，我也不知道如何去定义这种“滚烫的人生”，于是世人大多艳羡的光鲜体面，各种名次头衔和荣誉被规划进了我的短期目标。

可能是内心不够强大不足以支撑自己的信念，很容易因为别人的看法影响自己的情绪，去轻易地否定自己或者沾沾自喜。

这本书过段时间（我也不知道多久）我还会再过来看的。希望那时的自己能对这句话有稍微深一点点的体会。

《我们仨》

Posted on 2019-08-26 Edited on 2019-11-21 In 读书笔记

好像自从上了大学？或者是更早的从高中选了理科开始，就很少读书了。这里的书当然指的是那些无关应试无关技术，能够让人在读的过程中感受到愉悦发人深思的书了，再加上回想这么多年来，接收的观点绝大多数来自互联网，消磨时光的方式是手机，偶有停下来思考总结的时候猛然发现自己什么东西都没留下，东西学进去了多少不知道，痕迹是一点都没留下。用文字图片或视频去记录，稳定持续的输出也算是对自己内心的清扫和过去时光里陪伴自己的人和事物的一份交代吧。

《我们仨》这本书相对来说比较简短，其中有一些细节是我比较有印象的：

吃完饭后杨绛先生会和丈夫一起散步，他们专挑不熟悉的路走，称之为“探险”。坚信一家人在一起就能克服困难，无论条件多么艰苦都不会忘记工作和读书，还有早上的一杯红茶牛奶。读这本书就像身旁有一个和蔼的老人，在安静的午后将她一生的活美好或苦涩的经历娓娓道来，温馨且朴实，文人的生活并非只有诗词歌赋，同样需要关心吃饭睡觉，这本书更像是杨绛先生写给自己看的关于对丈夫女儿的怀念，没有华丽的辞藻更无关功利。

这是我最喜欢的一句话

“人间没有单纯的快乐。快乐总夹带着烦恼和忧虑。”
